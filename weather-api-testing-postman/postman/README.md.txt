# Weather API Testing (Postman)

This repository contains Postman collections for automated API testing of:

- OpenWeather API (Current Weather & 5-Day Forecast)
- Open-Meteo Historical Weather API

## Technologies & Tools
- Postman
- Environment & Collection Variables
- Pre-request Scripts
- JSON Schema Validation
- Postman Collection Runner

## Structure
- `/postman/collections` – Postman collections
- `/postman/environments` – Postman environments (API keys removed)

> API keys are not included for security reasons.
